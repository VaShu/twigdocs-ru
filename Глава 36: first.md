Фильтр ```first``` возвращает первый элемент последовательности (массива), строки или хэша:

```twig
{{ [1, 2, 3, 4, 5]|first }}
{# выведет 1 #}

{{ {a: 1, b: 2, c: 3}|first }}
{# выведет 1 #}

{{ "1234"|first }}
{# выведет 1 #}
```

Фильтр так же работает с объектами, которые реализуют ```Traversable``` интерфейс.
