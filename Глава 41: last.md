Фильтр ```last``` возвращает последний элемент последовательности - массива, хэша или строки:

```twig
{{ [1, 2, 3, 4]|last }}
{# выведет 4 #}

{{ {a: 1, b: 2, c: 3}|last }}
{# выведет 3 #}

{{ "12345"|last }}
{# выведет 5 #}
```

Фильтр так же работает с объектами, которые реализуют интерфейс ```Traversable```.
