Функция ```random``` возвращает случайно значение в зависимости от типа переданных параметров:

- случайный элемент массива
- случайный символ из строки
- случайное значение от 0 до указанного верхнего значения (включительно)

```twig
{{ random(['apple', 'orange', 'citrus']) }}
{{ random('ABC') }}
{{ random() }} {# работает как PHP-функция mt_rand #}
{{ random(5) }}
```

###### Аргументы

- ```values```: значения
